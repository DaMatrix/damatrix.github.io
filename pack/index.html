DaPorkchopPack

Info

The DaPorkchopPack is a semi-lightweight modpack that runs equally well on laptops and higher-end computers, has a dedicated server, and is under heavy development. I will be uploading videos of the pack on my YouTube channel, and you can keep up-to-date by joining the modpack's Discord server!

Installation

How to join the server

Once you get the modpack installed, simply click on "Multiplayer" and the server will already be there. How cool is that?

If for any reason the server isn't there, you can add it with the ip "play.pack.daporkchop.tk".
